# Управление аватарками фейков через админ-панель

## Цель
Убрать необходимость запускать отдельный скрипт для подготовки аватарок. Админ загружает фото прямо в Telegram-боте, выбирая категорию (мужские / женские / нейтральные). Также улучшить разнообразие имён и приблизить возраст фейков к возрасту текущего пользователя.

## Что меняем

### 1. Хранилище аватарок
- Новая таблица `fake_avatars`:
  - `avatar_id INTEGER PRIMARY KEY AUTOINCREMENT`
  - `gender TEXT NOT NULL` — `male`, `female` или `neutral`
  - `file_id TEXT NOT NULL` — Telegram `file_id`
  - `created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP`
- Удалить/оставить `scripts/prepare_fake_avatars.py` и `data/fake_avatars/` как резервный ручной вариант? Нет — удаляем неактуальный скрипт и папки аватарок, чтобы не путать.

### 2. Админ-панель: загрузка аватарок
- В меню «🎭 Фейки» добавить кнопку «🖼 Аватарки фейков».
- При нажатии показываются три кнопки:
  - «Мужские"
  - «Женские"
  - «Нейтральные"
  - «↩️ Назад"
- Выбор пола переводит админа в состояние `AdminMenu.fake_avatar_upload` и запоминает выбранный `gender` в `FSMContext`.
- Бот пишет: «Отправляй фото. Добавлено: 0. Когда закончишь, нажми Готово.»
- Под сообщением inline-клавиатура: `[Готово] [Отмена]`.
- При получении фото:
  - Берётся `message.photo[-1].file_id`.
  - Запись добавляется в `fake_avatars`.
  - Бот отвечает/редактирует счётчик: «Добавлено: N».
- При нажатии «Готово» состояние очищается, админ возвращается в меню фейков.
- При нажатии «Отмена» уже добавленные фото остаются, сессия закрывается.

### 3. Генератор фейков: выбор аватарки
- `pick_avatar_file_id(gender)` делает запрос к `fake_avatars` с фильтром по `gender` (fallback на `neutral`, затем на любую).
- Если в БД нет аватарок — возвращает `None`, фейк без фото.

### 4. Разнообразие имён
- Добавить пулы неформальных имён:
  - Мужские: Некит, Саня, Димон, Лёха, Пашка, Макс, Артёмка, Кирюха, Миша, Ваня, Егорка, Матвей, Тёма, Рома, Влад, Павлуша и т.д.
  - Женские: Лиска, Машка, Настюха, Вика, Катя, Соня, Даша, Алиса, Вероника, Поля, Лиза, Ксюша, Саша, Оля, Таня, Юля и т.д.
- В `_pick_name(gender)` с вероятностью ~40% выбирается неформальное имя, иначе полное.

### 5. Возраст фейков
- `_pick_age(viewer)`:
  - Целевой возраст = `viewer["age"]` ± случайное число от 0 до 3.
  - Затем обрезается снизу `max(filter_min_age, 16)` и сверху `min(filter_max_age, 100)`.
  - Если диапазон схлопнулся — возвращается граница.

## Тестирование
- Проверить создание таблицы `fake_avatars`.
- Проверить добавление аватарки через хендлер (mock сообщение с фото).
- Проверить выбор аватарки по полу/fallback.
- Проверить, что возраст фейка в ±3 от возраста пользователя и в рамках фильтров.
- Проверить, что неформальные имена могут появляться.
